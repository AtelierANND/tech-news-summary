以下のはてなブックマークのテクノロジーカテゴリ記事リストから、15記事を選定してください。

# 対象読者
Web制作会社に勤務する以下の職種の方々:
- ディレクター
- デザイナー
- フロントエンドエンジニア

# 選定基準と記事配分
以下の配分で15記事を選定してください：

1. **生成AI活用（5記事程度）**:
   - 業務での生成AI活用事例
   - AI支援ツール紹介
   - AIを使った開発・デザイン効率化
   - ChatGPT、Claude、GitHub Copilot等の活用法

2. **フロントエンド技術（4記事程度）**:
   - HTML/CSS/JavaScript/TypeScript
   - React/Vue/Next.js等のフレームワーク
   - パフォーマンス最適化
   - Webアニメーション

3. **デザイン領域（3記事程度）**:
   - UI/UXデザイン、デザインツール、デザインシステム
   - Figma、Adobe XD等のツール活用
   - ビジュアルデザイン、タイポグラフィ
   - アクセシビリティ、ユーザビリティ

4. **ディレクター・PM向け（3記事程度）**:
   - プロジェクトマネジメント
   - チーム運営、コミュニケーション
   - 非エンジニア向けテクノロジー解説
   - Web制作ワークフロー

# 除外基準
- Togetterなどのまとめ記事
- 極端に専門的すぎる記事（バックエンド、インフラ、低レイヤーなど）
- 特定の高度な専門知識を前提とする記事

# 出力形式
JSON配列で15記事を返してください。各記事は以下の形式:
```json
{
  "title": "記事タイトル",
  "url": "記事URL"
}
```
